How to install LaTeX locally on your PC

Created on June 11, 2025

2025   ·   latex   miktex   ·   tutorials

  • Installation
  • Installing Packages
  • Writing and Compiling a Project.

Hi, today I will describe how I installed MiKTeX, a TeX distribution, locally on my PC, along with my usual go-to workflow. Often, I faced issues regarding compile time on Overleaf. Hence, why not locally work with LaTeX?

So, let’s see the steps to install MiKTeX. I will also show how we can install packages, followed by a simple script to compile our project with multiple files without flooding our project directory.

Installation

  1. Install python. (not needed for MiKTeX, but it is useful for the script I will be sharing).
  2. Install VSCode (I prefer this, but any other editor would work).
  3. Go to MiKTeX and choose your OS and download.
  1. Follow the steps at https://miktex.org/howto/install-miktex, they are pretty self explanatory.
  2. Once MiKTeX is installed, you can search for MiKTeX Console on the Windows search bar and open it as administrator.
  1. Open Command Prompt, and type where pdflatex. If it shows a valid path under MiKTeX, your TeX Distribution has been installed!

Installing Packages

Under the Packages tab, you can easily search for a package and install it.

Writing and Compiling a Project.

I will be using a sample project (My Intro to Quantum Physics Assignment 2). It contains subfiles and images so that it would be a good demonstration.

  1. Open VSCode and create your project.
  1. Create a file called compile.py.
  2. Now, all you need to do is open the terminal (press Ctrl + `), and type in compile.py main.tex. Press Enter if any known errors come up. MiKTeX may also prompt you to install packages if needed. The compilation will continue.
  3. Now, main.pdf will be saved in your project directory. You can also open the pdf in split screen view in VSCode to get more of an overleaf feel.
